Для добавления нового языка необходимо:
 - выбрать ГО "LocalizationManager"
 - в "Language name" назвать язык
 - в "Datas" добавить перевод
 - в "Play" режиме нажать на кнопку "Space"

При выборе языка система сохраняет выбранный язык.

В локализации продумана система ID. Работает таким образом:
 - Находим объект с компонентом "Text"
 - Добавляем на него скрипт "TranslatorObject"
 - Добавляем объект в "LocalizationManager" в поле "Objects"
!!!!Главное условие!!!!
 - Номер элемента который необходимо переводить должен соответствовать номеру элемента перевода
